WORKFLOW

Jenkins Job triggers JMeter test execution.
JMeter runs tests, collects data (response.request.errors).
If assertions are not as expected, JMeter can raise a bug in Jira using the REST API.
Upload the test results to Jira using the Xray plugin.
In Jira, defects can be created automatically or linked to the relevant test cases.
